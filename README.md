## üöÄ Introduction

ConvoXR is a modern video conferencing platform built using **Next.js** and **TypeScript**.  
It allows users to securely authenticate, create meetings, schedule calls, and interact in real time using audio and video streaming.

This project was built as a **learning-focused full-stack application** to understand real-time communication, authentication, and scalable frontend architecture.
